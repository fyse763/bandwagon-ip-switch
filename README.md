## 先确认你的 IP 是真被封了，还是别的问题

这一步很多人跳过，然后花了钱换了IP发现根本不是IP的问题。

**判断方法**：用 ping.pe 这个工具，输入你的服务器IP，从中国大陆多个城市节点同时去ping。如果结果全红，几乎所有节点都超时，那基本可以确认 IP 被封了。如果只是部分城市不通，或者国内不通但国外通，那也符合被墙的特征。

相反，如果你 ping 完发现各地都能通，那问题可能在端口、在你的软件配置、在本地网络，换IP没用。

## 换IP之后还是连不上？几个排查思路

有时候付了钱、换了IP，连上去发现还是有问题。这种情况遇到过几次，原因通常是这样的：

**DHCP 没刷新**。搬瓦工把新IP分配给你了，但你本地客户端还在用旧IP连。解决方法是重新启动 VPS 一次，或者手动更新一下网络配置。通常重启就够了。

**端口被封而不是IP被封**。换IP解决不了端口封锁的问题。如果新IP也连不上，试一下换端口，看看是不是端口的问题。

**新IP还在生效中**。系统显示换好了，但实际的路由生效需要一点时间。等十几分钟再试一次。

还是不行的话，直接联系客服开工单。搬瓦工客服响应不算慢，工作时间内一般几十分钟内有回复。既然换IP的钱付了，让他们确认一下没问题是应该的。
